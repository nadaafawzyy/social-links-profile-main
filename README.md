# Social Links Profile

A clean, responsive **social links profile page** built from the *Frontend Mentor* challenge.

## 🚀 Live Demo
(https://nadaafawzyy.github.io/social-links-profile-main/)


## 📌 Overview
 ⚠️ **Note:**  **portfolio sections** are currently *added as likes* in navigation, but they are **not available yet** (work in progress).

This project presents a minimal profile interface with links to your social accounts.  
Users should be able to:

- See your profile photo and name
- View all your social links
- Interact with links (hover and focus styles)

## 🧠 Built With

- HTML5  
- CSS3  
- Responsive design

## 📁 Features

✔️ Clean layout  
✔️ Mobile‑first and responsive  
✔️ Easy customization of links and styles  
✔️ Simple and project‑ready
